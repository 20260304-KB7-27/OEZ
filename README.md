## 🎮 게임 진행 방식
1. 화면 중앙 상단에 퀴즈 문제와 제한 시간(5초)이 주어집니다.
2. 플레이어는 **키보드 좌우 방향키(⬅️ ➡️)**를 조작하여 노진구 캐릭터를 움직입니다.
3. 제한 시간 안에 정답이 적힌 왼쪽 또는 오른쪽 문 앞으로 이동하면 정답으로 판정됩니다.

<br>

## 🖥 화면 구성
프로젝트는 크게 4가지 상태의 화면으로 구성되어 있습니다.
1. **시작 페이지**: 게임 제목과 함께 '게임 시작' 버튼이 제공되는 인트로 화면.
2. **게임 페이지**: 5초 타이머, 퀴즈 말풍선, 정답이 적힌 두 개의 문, 그리고 조작 가능한 캐릭터(진구)가 배치된 메인 플레이 화면.
3. **결과 페이지 (게임 성공)**: 모든 퀴즈를 통과했을 때 나타나는 화면으로, 최종 점수 출력 및 성공 팡파르 효과음 재생.
4. **결과 페이지 (게임 실패)**: 오답을 선택하거나 시간 초과 시 등장. 퉁퉁이 엄마의 불호령과 함께 '다시 하기' 버튼 제공.

## 📂 퀴즈 카테고리
본 게임은 웹 개발의 기초 및 CS 지식을 다루는 4가지 주제로 구성되어 있습니다.

- **HTML (이채린)**: 웹페이지 텍스트 제목 태그, 하이퍼링크 태그, 이미지 삽입 태그의 올바른 사용법
- **CSS (석정한)**: 폰트 크기 상대 단위(em/rem), 배경(background) 단축 속성의 특징, 박스 모델(Box Model)의 시각적 범위
- **JavaScript (오지연)**: 변수와 상수 선언 키워드, 엄격한 동등 연산자(===), 비동기 처리 객체(Promise)의 개념
- **Protocol (이종원)**: 인터넷 기본 통신 프로토콜(TCP), IP 주소 체계(IPv4/IPv6), URL 구조 식별(Host 등)
